L1 — Anatomy of an Atom★☆☆

1. The centre of an atom is called the nucleus.

2. Protons (+) and neutrons (0) live inside the nucleus.

3. Electrons (-) orbit the outside in shells.

4. The nucleus is heavy; it holds almost all the atom's mass.

5. Electrons are so tiny they have almost zero mass/weight.

6. An atom is mostly made of empty space.

L2 — The Periodic Table Square★☆☆

1. The top number is the atomic number.

2. The 1 or 2 big letters represent the chemical symbol.

3. The bottom number is the atomic mass.

4. Atomic mass is usually a decimal but we round it to a whole number.

5. Helium's rounded atomic mass is 4.

6. The atomic number identifies which element it is.

L3 — Numeracy: The APE MAN Rule📊

1. A neutral atom has the same number of protons and electrons.

2. The atomic number tells you the number of protons.

3. To find neutrons, subtract the atomic number from the Mass.

4. If Mass = 19 and Atomic # = 9. Neutrons = 10.

5. If Mass = 40 and Atomic # = 18. Neutrons = 22.

6. If an atom has 7 protons, it also has 7 electrons.

L4 — Element Profile: Carbon★★☆

1. Carbon's atomic number is 6.

2. Carbon's atomic mass (rounded to whole) is 12.

3. A=P=E: Carbon has 6 protons and 6 electrons.

4. M-A=N: Mass (12) - Atomic (6) = 6 neutrons.

5. The symbol for Carbon is C.

6. Carbon is the essential element for all life/living things.

L5 — Element Profile: Sodium★★☆

1. Sodium's atomic symbol is Na.

2. Sodium's atomic number is 11.

3. The mass number rounds to 23.

4. A=P=E: Sodium has 11 protons and 11 electrons.

5. M-A=N: It has 12 neutrons (23 - 11).

6. Sodium forms table salt when combined with chlorine.

L6 — Metals vs Non-Metals★★☆

1. Metals are found on the left side of the table.

2. Non-metals are mostly on the right side.

3. They are separated by a zigzag line called the staircase.

4. Elements touching the line are called metalloids.

5. Sodium (left side) is a metal.

6. Exception: Hydrogen (top left) is actually a non-metal.

L7 — Master Profile: Oxygen★★★

1. Oxygen's rounded mass number is 16.

2. Number of Protons: 8.

3. Number of Electrons: 8.

4. Number of Neutrons: 8.

5. Oxygen is found on the far right, so it is a non-metal.

6. Is Oxygen's nucleus heavier or lighter than Carbon's? Heavier

L8 — Short Answer: Particles★★★

Explain how to calculate the number of neutrons in any element:

First, take the atomic mass

and round it to a whole number.

Then, subtract the atomic number.

The answer gives you the number of neutrons.

This works because the mass = protons + neutrons.

L9 — Element Profile: Iron★★☆

1. The atomic symbol for Iron is Fe.

2. Iron's atomic mass rounds up to 56.

3. Number of Protons: 26.

4. Number of Electrons: 26.

5. Number of Neutrons (56 - 26): 30.

6. Iron is in the middle of the table, so it is a transition metal.

L10 — Element Profile: Chlorine★★☆

1. Chlorine's atomic number is 17.

2. Chlorine's mass number rounds to 35.

3. Number of Protons: 17.

4. Number of Electrons: 17.

5. Number of Neutrons: 18.

6. Chlorine is on the far right, so it is a non-metal.

L11 — Numeracy: Missing Info📊

1. Since Protons = 9, the atomic number is 9.

2. Since A=P=E, the number of Electrons is 9.

3. To find the mass, we add Protons + Neutrons.

4. The Atomic Mass of Element X is 19.

5. Look at a periodic table. Element #9 is Fluorine.

6. The symbol for this element is F.

L12 — Numeracy: Heavy Elements📊

1. Gold's atomic number is 79.

2. Gold's atomic mass rounds to 197.

3. Number of Protons: 79.

4. Number of Electrons: 79.

5. Subtraction: 197 - 79 = 118.

6. Gold has 118 neutrons in its nucleus.

L13 — Key Vocabulary★☆☆

1. Atomic Number: Tells you the number of protons.

2. Atomic Mass: The combined weight of protons and neutrons.

3. Symbol: The 1 or 2 letter abbreviation for an element.

4. Nucleus: The heavy center of the atom where protons live.

5. Shells: The outer rings where electrons orbit.

6. Metalloids: Elements touching the zigzag staircase.

L14 — Master: The Mystery Element★★★

1. The element has 20 protons. Its atomic number is 20.

2. The element has 20 neutrons. Its mass number is 40.

3. A neutral atom of this element has 20 electrons.

4. Look at a periodic table. Element 20 is Calcium.

5. The chemical symbol is Ca.

6. It is found on the far left side, so it is a metal.
