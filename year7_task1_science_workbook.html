Part A — Calculating Means & Spotting Outliers

The mean (average) smooths out small variations between trials. It is the most useful single number to represent your results.

Formula: Mean = (Sum of all valid values) ÷ (Number of trials used)
⚠️ Always check for outliers BEFORE calculating the mean. Exclude them and record why.
📝 Worked Example
A student records these force values at 30°: 3.8 N, 4.0 N, 3.9 N, 8.5 N, 3.7 N

Step 1 — Spot the outlier: 8.5 N is much higher than the rest → likely a measurement error → excluded.
Step 2 — Calculate the mean: (3.8 + 4.0 + 3.9 + 3.7) ÷ 4 = 15.4 ÷ 4 = 3.85 N

Part B — Evaluating Reliability & Validity

Concept What it means How to improve it Evidence from YOUR experiments
Reliability Results are consistent across repeated trials — similar values each time Increase number of trials; use same person to do each measurement; use precise equipment
Validity The experiment actually tests what it claims — only one variable changed at a time Control all variables except the independent variable; use a control experiment
Accuracy Results are close to the true value — equipment is calibrated and used correctly Zero the spring balance before each use; read scale at eye level; take measurements carefully
