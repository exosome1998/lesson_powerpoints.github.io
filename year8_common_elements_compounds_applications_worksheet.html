1. Oxygen has the symbol O.

2. Oxygen gas is written as O₂.

3. O₂ has two oxygen atoms.

4. Humans use oxygen for breathing.

5. Medical oxygen is used in some hospitals.

6. True or False: O₂ is made from oxygen atoms only. True

1. Water is made from hydrogen and oxygen.

2. The formula for water is H₂O.

3. H₂O has two hydrogen atoms and one oxygen atom.

4. Water is used for drinking and cleaning.

5. Hydrogen has the symbol H.

6. True or False: Water is a compound. True

2. The formula for carbon dioxide is CO₂.

3. CO₂ is a gas at room temperature.

4. Plants use carbon dioxide in photosynthesis.

5. CO₂ can make fizzy drinks bubbly.

6. True or False: CO₂ contains calcium. False

2. Sodium has the symbol Na.

3. Chlorine has the symbol Cl.

4. The formula for table salt is NaCl.

5. Salt is used to flavour and preserve food.

6. True or False: NaCl contains sodium and chlorine. True

3. It is found in limestone, shells and chalk.

5. Calcium has the symbol Ca.

6. True or False: Calcium carbonate is a compound. True

4. Aluminium is used for cans and foil.
