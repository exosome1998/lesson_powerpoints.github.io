Year 12 Software Engineering — Code Input/Output Lab

24 runnable code and SQL questions: 6 per module. Students edit code, run it, check output, and reveal sample solutions.

Runtime note: JavaScript runs immediately in the browser. Python and SQL use Pyodide from a CDN, so they need internet access the first time. SQL uses Python's sqlite3; Python standard-library imports such as hashlib, json, math, re and statistics are supported.
